# 🍽️ RatriNukkad – Food Delivery App

**RatriNukkad** is a responsive food delivery web application designed to provide **anytime, anywhere** food ordering with a smooth and user-friendly experience. The project focuses on late-night food availability, quality service, hygiene, and efficient delivery using modern web technologies.

---

## 📌 Project Overview

RatriNukkad is developed as a web-based food delivery platform that allows users to browse food items, place orders, and interact with the system through a clean and responsive interface. The project is built using **HTML, CSS, and JavaScript**, with emphasis on usability, performance, and scalability.

---

## 🎯 Objectives

- Provide a seamless online food ordering experience  
- Support late-night food delivery concept  
- Ensure responsive design for mobile and desktop users  
- Improve customer experience through clean UI/UX  
- Lay a foundation for future backend and payment integration  

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the web pages  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript** – Client-side interactivity and logic  

---

## ✨ Features

- Responsive and modern UI design  
- Attractive homepage with hero section  
- Food menu display  
- Book a Table feature  
- Contact and social media integration  
- Navigation bar and footer sections  
- Favicon and branding elements  
- Optimized for desktop and mobile devices  

---

## 🚀 How to Run the Project

1. Download or clone the repository  
2. Open the project folder  
3. Run `index.html` in any modern web browser  
4. No additional installation required
